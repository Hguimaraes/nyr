Data Science Journey



Unfortunately this is not "An unexpected Journey" because is something I've be planing for a long time but was waiting for the right time. The "Data Science Journey" is the main resolution for 2018 and the one that I'm going to focus more. It's a collection of steps to study and be more activate in the Data Science field focusing on Machine Learning. I hope to write on my blog about every step finished here. Enjoy!

Math background

The basic foundation of Data Science and Machine Learning is Math. You probably can achieve good results without understanding every details on how an algorithm is working but to push through you can't avoid have a solid foundation here. As an undergrad in Computer Engineering for my luck most part of these topics are not new but I really need to refresh some concepts and this is the first part of this journey.

  • Single Variable Calculus
  • Multivariable Calculus
  • Linear Algebra and Functional Analysis
  • Theory of Probability and Statistics
  • Non-convex Optimization
  • Information Theory

Data Science, Machine Learning and Deep Learning

This is an extensive part and it's not possible to go deep in each topic. Here I want to focus on some important parts (Such as Exploratory Data Analysis, Statistical Learning, NNs and CNNs) and go superficially in the others, just to understand the basics and know what people are doing on those fields.

  • Cleaning Data and Exploratory Data Analysis
  • Introduction to Natural Language Processing
  • Introduction to Statistical Learning
  • Neural Networks
  • Convolutional Neural Networks (CNN)
  • Recurrent and Recursive NN
  • Autoencoders
  • Deep Generative Models
  • Reinforcement Learning

Kaggle and Portfolio

And a really important part for me is to join more deeply the Kaggle community. In the middle of 2017 I had an extra time and could dedicate more time to this and saw how fast I could learn from other experienced users and this is awesome! Also using Kaggle (and Github) is a good way to show your work to other people and be recognized for it.

  • Get the "Expert" ranking in Kaggle Competitions
  • Won at least one Silver medal in a Kaggle Competition
  • Develop a Computer Vision Project using CNNs on Github
  • Develop a Project using RNNs on Github
  • Develop a Project using RL on Github
  • Develop a Big data Project using Spark on Github
  • Make a LinkedIn post about some of those projects
